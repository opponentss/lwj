关于本站

一个分享技术、生活和思考的个人空间

网站简介

这是一个使用 Django 构建的个人博客/导航网站，旨在分享技术文章、生活感悟和实用工具。网站设计简洁、响应式，注重用户体验。

灵感来源于 lvyovo-wiki.tech，采用了类似的卡片式布局和功能模块。

技术栈

  • 后端： Django 4.2 + SQLite/PostgreSQL
  • 前端： HTML5 + CSS3 + 原生 JavaScript
  • 编辑器： django-markdownx (Markdown 支持)
  • 部署： 支持 Docker、Nginx、Gunicorn
  • 其他： Font Awesome 图标、Google Fonts 字体

功能特性

文章系统

支持 Markdown 编辑和渲染，包含分类、标签、阅读量统计

日历组件

动态日历，高亮有文章发布的日期，支持点击跳转

随机推荐

每次刷新或点击"换一批"随机获取推荐文章

数据统计

显示文章总数、访问量等统计数字

响应式设计

适配桌面、平板和手机等不同屏幕尺寸

权限管理

管理员可发布文章，普通用户仅可浏览
